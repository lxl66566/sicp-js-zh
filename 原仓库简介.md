## 下载

### Docker

```
docker pull apachecn0/sicp-js-zh
docker run -tid -p <port>:80 apachecn0/sicp-js-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install sicp-js-zh
sicp-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g sicp-js-zh
sicp-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```
